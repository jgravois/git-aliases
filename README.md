
# git-aliases

Commonly used git aliases. Fork to add more!

## Installation

```bash
$ source aliases.sh
```

## Aliases

| Alias | Command |
|-------|---------|
| `gs`  | `git switch` |
| `gsc`  | `git switch -c` |
| `gsm`  | `git switch master` |
| `gstat`  | `git status` |
| `gd`  | `git diff` |
| `gdc`  | `git diff --cached` |
| `gl`  | `git log` |
| `gpom` | `git pull origin master` |
| `gpum` | `git pull upstream master` |
| `gf` | `git fetch -all` |
| `gp`   | `git push` |
| `gpuo`   | `git push -u origin` |
| `ga`   | `git add .` |
| `gm`   | `git commit -m` |
| `gam`  | `git add . && git commit -m` |
| `gb`   | `git branch` |
| `gbd`   | `git branch -D` |
| `gco`  | `git checkout` |
| `gcom`  | `git checkout master` |
| `gcob`  | `git checkout -b` |
| `gc`  | `git clone` |
| `gr`  | `git rebase` |
| `grm`  | `git rebase master` |
| `grc`  | `git rebase --continue` |
| `grs`  | `git rebase --skip` |
| `gcp` | `git cherry-pick` |
